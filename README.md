# Apple Music TUI (Python)

Simple, modern terminal controller for Apple Music using Python and AppleScript.

## Requirements
- macOS with the Music app
- Python 3.9+
- Terminal with mouse/keyboard support

## Run
```bash
cd /Users/liandiwijaya/dev/apple/musictui
python3 apple_music_tui.py
```

If macOS prompts for Automation access, allow your terminal to control Music.

## Controls
- Up/Down or j/k: move selection
- Enter: play selected playlist
- Space: play/pause toggle
- o: play
- a: pause
- s: stop
- n: next track
- p: previous track
- r: refresh playlists + now playing
- q: quit
- Mouse: click buttons in the control strip
